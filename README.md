# Access8Math Web Template

## Prerequisites

- Node.js >= 18
- NPM >= 9

## Template Release

Please tag the repository without a Release. One action will handle the Release work.

## Package Release

Manually trigger the github action "Publish package to GitHub Packages".

## Project setup

```
npm install
```

### Development

```
npm runs start
```

### Build for Production

```
npm run build
```

### Lint

```
npm run lint
```

### Test

```
npm run test
```
