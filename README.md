# Access8Math Web Template

## Prerequisites

- Node.js >= 18
- NPM >= 9

## Template Release

Please tag the repository without a Release. One action will handle the Release work.

## Package Release

Tag "package-release" to trigger the pacakge release process.

## Project setup

```
npm install
```

### Development

```
npm runs start
```

### Build for Production

```
npm run build
```

### Lint

```
npm run lint
```

### Test

```
npm run test
```
